mscctran: Celestial coordinate transformation using plate solution
==================================================================

**Package: msctools**

.. raw:: html

  <section id="s_synopsis">
  <h3>Synopsis</h3>
  <p>
  Transform between pixel (logical) and celestial (world) coordinates
  based on an input image.  The transformation may use just the linear
  coordinate transformation in the image header or a plate solution
  from a database file with the file and entry specified in the image
  header keyword WCSSOL.
  </p>
  </section>
