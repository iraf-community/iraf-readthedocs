mscfinder: Package to do astrometry on mosaic data
==================================================

**Package: msctools**

.. raw:: html

  <section id="s_synopsis">
  <h3>Synopsis</h3>
  <p>
  This package is currently intended for use only by instrument support
  personnel to determine plate solutions for Mosaic data.  It is based
  on an interim tool derived from the <b>finder</b> package that
  takes input from a simple coordinate file and Mosaic images.
  </p>
