  <h3>Description</h3>
  <p>
  The spfitpkg contains a set of programs useful for fitting complex UV and
  optical spectra.  The central task is <span style="font-family: monospace;">"specfit"</span>, and the tasks <span style="font-family: monospace;">"dbcreate"</span>
  and <span style="font-family: monospace;">"dbcheck"</span> are useful utilities for creating and checking the input
  database files used by specfit for the parameters of he fit.  See the
  help files for these tasks for further details.
  </p>
