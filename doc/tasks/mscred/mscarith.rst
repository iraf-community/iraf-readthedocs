mscarith: Image arithmetic mosaic exposures
===========================================

**Package: mscred**

.. raw:: html

  <section id="s_synopsis">
  <h3>Synopsis</h3>
  <p>
  This tasks performs image arithmetic, using <b>imarith</b>, by expanding
  each mosaic file into image extensions and creating an output mosaic file
  from all the results of operating on the extensions.
  </p>
  </section>
  <section id="s_usage_">
  <h3>Usage	</h3>
  <p>
  mscarith operand1 op operand2 result
  </p>

  <section id="s_restrictions">
  <h3>Restrictions</h3>
  <p>
  Image sections are not allowed in the mosaic filenames.  When two
  operand images are used the image extensions are not matched by extension
  name but only by order in the file.
  </p>
